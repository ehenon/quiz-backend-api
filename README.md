# NestJS + MongoDB API boilerplate

REST API template using NestJS and MongoDB, with:
- sample resource ("instrument")
- health route
- lint
- unit tests
- CI workflow
- custom JSON logger
- `.env` configuration system
- OpenAPI spec
- Conventional commits spec

## Available commands

```
npm run start:dev
npm run start:debug
npm run lint
npm run test
npm run build
npm run start
npm run start:prod
```
